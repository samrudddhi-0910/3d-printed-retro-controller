# Case Study: 3D Printed Retro Controller Form Study

This repository serves as a rapid prototyping and 3D printing case study of a retro game controller shell. 

This was printed as a college lab exercise to analyze the physical ergonomics, print speeds, and structural tolerances of a rapid-use visual prototype. 

---

##Physical Prototype Result

The final print achieved a remarkably smooth, high-fidelity surface finish. It serves as a non-functional form study model to evaluate hand-feel and button layout.

## 🛠️ Print Settings & Technical Specifications

The model was analyzed and processed using the flashprint environment to optimize for rapid production without sacrificing structural integrity.

| Parameter | Specification |
| :--- | :--- |
| **3D Printer** | Flashforge Adventurer 5M Pro |
| **Nozzle Diameter**| 0.6 mm |
| **Print Time** | ~15 Minutes (Highly Optimized) |
| **Material Weight**| 14.77 g |
| **Filament Length**| 4.95 m |
| **Total Layers**   | 50 layers (Height: 14.98 mm) |
| **Print Profile**  | Normal Mode |

## 🔍 Key Observations & Prototyping Takeaways

* **The 0.6 mm Advantage:** Using a larger 0.6 mm nozzle allowed this model to finish printing in an incredibly fast **15 minutes** while maintaining robust walls.
* **Ergonomic Testing:** As a visual prototype, the scale is ideal for a pocket-sized device. The physical model allows designers to feel button placements (D-Pad and Action Buttons) in real-time.
* **Bed Adhesion:** A blue outer **brim** was sliced to secure the sharp, narrow edges of the controller grips, successfully preventing any warping or lifting during the high-speed print.
